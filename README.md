# MetaOffice

## 프로젝트 개요
- 메타버스 기반의 오피스/업무 환경 구축을 위한 프로젝트
- 다양한 업무 기능, 설계 문서, 요구사항 정의서 등 포함

## 주요 파일 및 문서
- 기능명세서.xlsx: 전체 기능 명세
- 메타버스 SW 설계.emmx: 소프트웨어 설계 문서
- 메타버스 기능 목록.emmx: 기능 목록 및 구조
- 메타버스 오피스 개발 표준 정의서.hwp: 개발 표준
- 메타버스 오피스 요구사항 정의서.hwp: 요구사항 정의

## 특징
- 실제 코드보다는 설계, 명세, 표준, 요구사항 등 문서 중심
- 메타버스 오피스 시스템의 체계적 개발을 위한 기반 자료 제공

## 실행 방법
- 별도의 실행 코드 없음 (문서 참고)

## 참고
- 문서 파일은 전용 뷰어(엑셀, 한글, XMind 등) 필요
